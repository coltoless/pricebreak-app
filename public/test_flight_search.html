Flight Search Interface Test

Test Instructions:

  1. Go to http://localhost:3000/ to test the main interface
  2. Try typing "JFK" in the From field - you should see airport autocomplete
  3. Try typing "LAX" in the To field - you should see airport autocomplete
  4. Click on the date fields to select departure and return dates
  5. Click on the travelers field to adjust passenger count and cabin class

  • ✅ Date pickers should work for departure and return dates

  • ✅ Swap button should exchange origin and destination
